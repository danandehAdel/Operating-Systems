=========================================================================
COMPILING
To build the cat program:
make

To clean up the temporary object files:
make clean
==========================================================================
RUNNING

To run the cat.c
./cat

Format of using the cat.c program
./cat input

Makefile
Make object file from .c file and combines the object file to make
an executable file to run the shell program.
==========================================================================
FUNTIONS IN CAT.C

System calls used in cat.c are open(),read(),write(),perror(), and close().
